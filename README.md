> Loopback Url Shortener Component

## Installation

```sh
$ npm install --save loopback-component-urlshortener
```

## Usage

```js
var loopbackComponentUrlshortener = require('loopback-component-urlshortener');

loopbackComponentUrlshortener('Rainbow');
```
